Overwatch 2's 9th Anniversary Event brings the biggest celebration yet with legendary skins, new game modes, and special events that honor the game's rich history while looking toward the future.

## Legendary Skins

The anniversary introduces:
- 9 new legendary skins (one for each year)
- Mythic skin for "Tracer: Chrono Guardian"
- Epic skins for all original heroes
- Weapon charms and player icons
- Highlight intros and emotes

## New Game Modes

Limited-time game modes include:
- "Mystery Heroes 2.0" with balanced team compositions
- "All-Star Showdown" featuring fan-favorite abilities
- "Time Attack" racing mode with hero abilities
- "Boss Battle" against giant Omnic enemies
- "Capture the Flag" competitive mode

## Special Events

The anniversary celebration features:
- Developer Q&A sessions
- Player art showcase
- Cosplay contests with prizes
- Charity fundraising events
- Community tournaments

## Rewards System

Players can earn:
- Free loot boxes for daily login
- Anniversary currency for event shop
- Exclusive player titles
- Animated sprays and voice lines
- Profile backgrounds and borders

The event runs for three weeks, with new content unlocking each week.
