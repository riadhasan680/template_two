Blizzard has announced significant changes to Overwatch 2's competitive mode for Season 10, introducing a new ranking system and quality-of-life improvements based on player feedback.

## New Ranking System

Season 10 introduces:
- 8-tier ranking system instead of 6
- Skill rating transparency improvements
- Performance-based bonus SR
- Decay prevention for active players
- Placement matches reduced from 5 to 3

## Map Pool Changes

The competitive map pool updates:
- New map: "Neon Tokyo" added
- Removed maps: "Hollywood" and "Temple of Anubis"
- Hybrid maps receive balance adjustments
- Push mode gets competitive integration

## Role Queue Updates

Role queue improvements include:
- Faster matchmaking times
- Better role balance algorithms
- Off-role rating display
- Flex queue option for casual competitive

## Rewards System

New competitive rewards:
- Exclusive weapon charms for top 500
- Animated sprays for Master+ players
- Golden weapons available earlier in the season
- End-of-season bonus based on performance

The changes go live with Season 10's launch in January 2025.
